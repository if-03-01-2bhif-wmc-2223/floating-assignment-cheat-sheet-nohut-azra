First Section


What is CSS Cascading Style Sheets (CSS) is a stylesheet language used
to describe the presentation of a document written in HTML or
XML (including XML dialects such as SVG, MathML or XHTML).
CSS describes how elements should be rendered on screen,
on paper, in speech, or on other media

properties of the element such as width and height of an image.
Attributes are always specified in the start tag (or opening tag)
and usually consists of name/value pairs like name = "value"

Use combinators properly: direct child, descendant, adjacent sibling, sibling This selector is used for the purpose of styling the adjacent siblings.
It is represented by using a plus (+) sign between the two elements.
As the name suggests, the child combinator selects all the children of an element.
It is different from the descendant selector in a way that it selects
only the direct children of an element. Furthermore, it is represented by a greater than (>) sign.
As the name suggests, this css combinator styles all the siblings of an
element that share the same parent. It represented by a tilde (~) sign.
For the purpose of selecting all the successors of an element,
the descendant selector is used. Successors include all the children of an element,
may it be a direct child or a child multiple levels down the DOM tree.
It is represented using a space between the parent and child element.
